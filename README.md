### 🚀 IDM Activator Debotz
## 🚀 NEW Update IDM 6.42. Build 63 [Realease](https://github.com/debotzzz/IDM_Activator/releases/tag/latest)

**IDM Activator Debotz** adalah aplikasi berbasis Python dengan antarmuka
GUI modern yang membantu proses aktivasi Internet Download Manager
secara otomatis.

Aplikasi ini memiliki tampilan modern, sistem pengecekan update
otomatis, serta beberapa fitur tambahan untuk mempermudah pengguna dalam
mengelola IDM.

------------------------------------------------------------------------

### ⬇️ Download
IDM Acktivator Debotz dapat diunduh dari [Download](https://github.com/debotzzz/IDM_Activator/releases/latest/download/IDM_Activator_Debotz.exe) dan jalankan.

### ✨ Fitur Utama

-   ⚡ Aktivasi otomatis IDM
-   🛑 Menutup proses IDM sebelum aktivasi
-   💻 Membuka IDM langsung dari aplikasi
-   📥 Download IDM
-   🔄 Auto update dari GitHub
-   📊 Status aktivasi dengan progress bar
-   🎨 Modern GUI berbasis Tkinter
-   🔐 Menggunakan hak akses Administrator

------------------------------------------------------------------------

### 🖥️ Tampilan Aplikasi

Aplikasi memiliki beberapa panel utama:

-   **Status Aktivasi**
-   **Informasi Sistem**
-   **Control Panel**

Menu utama yang tersedia:

-   Exit IDM
-   Aktivasi
-   Buka IDM
-   Download IDM
-   Check Update

-   Python 3.8+
